# yu-gi-boy

游戏王(Yu-Gi-Oh! DUEL LINKS) 挂机程序

# 声明
本项目用作学习,禁止用于商业出售目的,由此导致的问题,作者不负任何责任。

+ [Github项目地址](https://github.com/fangker/yu-gi-boy)
+ 该程序运行前请把游戏分辨率设置为 1024*576。
+ 游戏窗口前置后,不可进行覆盖或最小化，可以通过虚拟机运行。
